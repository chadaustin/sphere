RPG Spriteset File Format
Chad Austin
12.20.98


---- 128 byte header ----

byte signature[4];   // Must be ".rss"
word version;        // Must be 1
word num_frames;
word frame_width;
word frame_height;
word frame_bpp;
byte reserved[114];


---- Start spriteset data ----
The spriteset is just a serious of RGBA frames, each of which is frame_width *
frame_height * 4 bytes long.
