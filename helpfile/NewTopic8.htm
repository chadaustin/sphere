The Sphere Interface

The Sphere interface is alike most standard Windows 95/98/ME/2000 interfaces and is easy to get used to. Playing around with the Interface and the commands will help. The Sphere Editor has an MDI interface. This means that there is one main Editor window which can contain many smaller windows. In this case, the MDI window is the entire editor and example of the smaller windows are the project window and the script window. The MDI interface allows the developer to have the project, scipt and map editor windows all open at the same time.

The interface can be divided into three parts, the menu, right at the top of the window(directly below the title bar which reads "Sphere Development Environment"), the toolbar, directly below the menu and the application workspace, which contains the smaller windows.

The small windows in the application workspace can be manipulated(minimized, maximized and closed) by the small title bar row of options at the top right hand side of the window. When the small windows are maximized, the minimize, maximize and close buttons appear directly below the MDI window's minimize, maximize and close buttons. Take note that unlike all other windows only one project window can be opened at one time. These windows can also be manipulated with the Menu item Window.

Dialogs are dialog boxes which appear when you run a function, for example, you click New Project, a dialog box, which queries you about the Project Title and Game Name is displayed.
