The Font Menu

Resize Character

This will allow you to resize the size of the current font character. The default size is 8 x12.

Resize All Characters

This will allow you to resize the size of the all the font characters. The default size is 8 x12.

Simplify

This will simplify the font so only two colors are allowed. This is either opaque or transparent.

Generate Gradient

This will generate a gradient from one color to another. The option beside the color picker is the transparency changer, 255 is opaque and 0 is transparent.
