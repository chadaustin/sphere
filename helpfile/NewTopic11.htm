The Project Menu (Enabled when a project is opened or created)

Insert

Allows you to insert a new map, font, etc. into your project. Just choose what you want and then browse your computer for the map file, font file, spriteset file, etc.

Refresh

Refreshes the project info by reading the game.inf file again(I think so) includes any new content that you have added manually into the game.inf file.

Run Game

Builds and runs the game from within the editor using the engine.

Configure Sphere

Configure the dlls and your display drivers that are used to run Sphere. You can also configure the color depth of your screen that you want Sphere to run in. Remember, Sphere supports 32-bits.

Package Game!

Packages the game for distribution. A package is a collection of files rolled into one. Sphere can read these files like normal, except the files are from the package. These use a zlib like algorithm to compress the packages for distribution. These packages are good if you don't want an average computer user to be able to rip out your pictures and resources and scripts.
